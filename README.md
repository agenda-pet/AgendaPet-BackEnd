# Agenda-Pet

## Sobre o Projeto

O Agenda-Pet é um sistema desenvolvido para auxiliar a gestão de pet shops, permitindo o cadastro de clientes, pets, funcionários e serviços, além do gerenciamento de agendamentos e histórico de atendimentos.

O projeto foi criado como trabalho de final de semestre, aplicando conceitos de análise de requisitos, modelagem de sistemas, banco de dados, desenvolvimento web e gestão de projetos.

## Objetivo

Digitalizar e organizar os processos de atendimento de um pet shop, substituindo controles manuais por uma solução centralizada, segura e eficiente.

## Principais Funcionalidades

- Autenticação e controle de acesso de usuários.
- Cadastro e gerenciamento de funcionários.
- Cadastro e gerenciamento de clientes.
- Cadastro e gerenciamento de pets vinculados aos seus tutores.
- Agendamento, alteração e cancelamento de serviços.
- Histórico completo de atendimentos dos pets.
- Cadastro de serviços e preços.
- Busca de clientes por CPF e pets por nome.
- Registro de auditoria das alterações realizadas no sistema.

## Regras de Negócio

- Todo pet deve estar vinculado a um tutor.
- Não é permitido agendar dois pets para o mesmo funcionário no mesmo horário.
- O CPF do cliente deve ser único.
- Apenas administradores e gerentes podem cadastrar funcionários ou alterar preços.
- O histórico de atendimentos deve ser preservado mesmo após alterações ou inativação de cadastro.

## Requisitos Não Funcionais

- Interface responsiva para computadores, tablets e smartphones.
- Armazenamento seguro de senhas utilizando criptografia.
- Atualização rápida dos horários disponíveis para evitar conflitos de agendamento.

## Tecnologias Utilizadas

- CSS
- JavaScript
- Banco de Dados SQL
- API
